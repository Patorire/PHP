PHP

PHP es un lenguaje de programación que se utiliza para el desarrollo de aplicaciones y la creación de sitios web. Es fácil de usar y está en constante perfeccionamiento.
PHP es de código abierto y no tiene restricciones de uso vinculadas a los derechos.

¿Cuáles son las ventajas de PHP?

  • A día de hoy hay gran demanda de profesionales con experiencia en aplicaciones web y PHP es el recurso principal para esto.
  • El lenguaje PHP es considerado como uno de los más fáciles de aprender. Existe mucho material de apoyo que ofrece la comunidad de programadores que utilizan el lenguaje, mucho tutorial y folletos online.
  • Al ser de código abierto este no genera costos para los programadores que trabajan con él, siendo así una gran ventaja, especialmente para autónomos.
  • Además los programadores pueden mejorar aún más PHP usando el facilitador de código abierto, eliminando fallas, bugs o incluso se puede trabajar en la optimización del desempeño.
  • Una de las principales preocupaciones de los programadores es la complejidad que tendrán sus aplicaciones web, ya que cuantos más recursos tengan, más datos comenzarán a administrar. Los sitios que tienen gran cantidad de detalles visuales o otras cosas de gran porte, tienden a necesitar gran cantidad de información cuando estos están activos. Sin PHP sería complicado ejecutar este tipo de aplicaciones con el rendimiento mínimo requerido para la experiencia de navegación. En resumen a parte de facilitar el trabajo, hay grandes ventajas en cuanto a rendimiento.
  • PHP es compatible con diferentes bases de datos: Oracle, MySQL, SQLite, Sybase.

¿Pero para qué sirve PHP?

PHP está diseñado para incrustarse en html, es decir, un documento HTML puede contener elementos PHP integrados siempre cuando se sigan ciertas normas.
Al incrustarse en HTML este añade nuevas funcionalidades. HTML es un código estático que por lo tanto crea webs puramente estáticas, aunque el usuario interactue el código no variará y por lo tanto la página tampoco.
PHP cambia esto. El lenguaje conecta ciertos elementos de una página con determinados datos, permitiendo que el contenido cambie con las peticiones de los usuarios. Esta petición es mandada a un sevidor el cual a su vez esta conectado a una base de datos de donde se extrae información. Esta información vuelve a la página, la cual variará.
Esto se puede también hacer con JavaScript, la diferéncia es que la página que devuelve el servidor es estática, mientras que en JavaScript es dinámica.

Más acerca de PHP

Como se puede ver en la imagen de código el contenido del archivo incluye código HTML y en un momento dado se coloca la etiqueta ?php que indica el comienzo del código PHP, es por eso que decimos que el código PHP está incrustado en HTML.
Lo mejor de PHP es que es muy fácil para los principiantes y además ofrece muchas características avanzadas para los programadores profesionales. Con PHP se puede procesar la información de formularios, generar páginas con contenidos dinámicos, enviar y recibir cookies... Otra habilidad que tiene PP es que puede crear imagenes (gráficas) a partir de datos. PHP se puede enlazar con otros lenguajes muy potentes como lo es Java, por lo que son buenos amigos.

Módulos y Extensiones

PHP tiene una gran variedad de extensiones predefinidas y módulos que logran facilitar tareas como el manejo de XML, imagenes, la creación de PDF... Los módulos logran extender la funcionalidad básica de PHP.

Mantenimiento y Escalabilidad

PHP es adecuado a proyectos de diferentes tamaños. Generalmente los sitios web pequeños son construidos con PHP y las aplicaciones empresariales aprovechan patrones de diseño que aseguran un código mantenible y escalable.

¿Dónde comenzó PHP?

PHP es el sucesor de un producto llamado PHP/FI. Creado en 1994 por Rasmus Lerdorf, la primera aparición de PHP era un conjunto simple de ficheros binarios Common Gateway Interface (CGI) escritos en el lenguaje de programación C. Al principio lo utilizaba para rastrear visitas de su currículum online, este llamó al conjunto de scripts Personal Home Page Tools también llamado PHP Tools. Rasmus reescribió PHP Tools, produciendo una implementación más grande y rica. El nuevo modelo tubo la capacidad de interaccionar con bases de datos y mucho más, proporcionando un entorno de trabajo donde los usuarios prdrían desarrollar aplicaciones web dinámicas sencillas.
En junio de 1995 Rasmus publicó el código fuente de PHP Tools, permitiendo a desarrolladores usarlo como considerasen apropiado. Esto también permitió a los usuarios resolver erroes del código y mejorarlo.
En septiembre de 1995 Rasmus amplió PHP y cambió el nombre a FI (Forms Interpreter), la nueva implementación incluía algunas de las funciones básicas de PHP tal y como se conoce a día de hoy. Tenía variables como las de Perl, interpretación automática de variables de formulario y sintaxis incrustada HTML. Al mes siguiente todo esto empezó a cambiar, en octubre de 1995 Rasmus publicó una versión nueva del código, volviendolo a renombrar como PHP y en aquel entonces se presumia de su interfaz de scripts avanzada. El lenguaje buscaba parecerse a C en estructura, así sería más fácil para los desarrolladores familiarizados con C, Perl y lenguajes similares. Se implementó para sistemas UNIX y compatibles con POSIX, y viendo el potencial que podría tener para una implementación de Windows NT este se empezó a explorar para ello.
Aunque su vida de desarrollo fue corta PHP siguió creciendo en popularidad. En 1997 y 1998, PHP/FI tenía un culto de varios miles de usuarios en todo el mundo. Posteriormente se creo una version PHP 3, que fue organizada por Andi Gutmans y Zeev Suraski, los cuales preguntaron a Rasmus y para la implementación de este lenguaje. Finalmente Rasmus aceptó y se renombró como PHP. Después de esta versión PHP 3 salierón las versiones PHP 4 y PHP 5 que es la usada actualmente.
